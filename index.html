Reflection & Learnings

My experience on taking up this course, Managing Enterprise Servers, has been quite a hectic rollercoaster ride, if such feeling is compared to the feeling of being a student of this course. Basically, throughout this course, I learned a bunch of concepts and techniques in managing not just local servers, but remote ones as well.

I discovered the automated process of managing these remote servers which uses an open-source platform called Ansible. Majority of our activities used Ansible to automate the installation and configuration processes of applications and software through running a playbook that has a file extension of yml or yaml. Apart from this, I managed to grasp the concept of containerization through another open-source platform named Docker. Only a few activities related to containerization were done, so only a small percentage of it was practiced if to be compared to the activities accomplished with the application of Ansible. Lastly, as I reached the end of the course, the topic of cloud computing was discussed briefly. OpenStack is the open-source cloud computing platform that was discussed and applied in the last remaining activities for this subject.

A huge thanks to this course, I have another set of skills to add to my future resume or curriculum vitae once I have reached the stage of starting a career in the tech industry. There were of course memorable moments that I had while undergoing this course. I met my classmates for the first time in personal and was able to build new friendships with some of them. The idea of teamwork is what really made me enjoy and persevere in accomplishing the required tasks for the completion of this course. Perhaps, teamwork is really a necessary attribute in the engineering field, so I really appreciate that it was somehow practiced on this subject.
